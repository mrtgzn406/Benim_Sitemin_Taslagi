Web Siteleri

Sizin için web siteleri tasarlayabilirim. Örneğin kişisel web sitesi, e-ticaret sitesi, blog sitesi, forum sitesi vb. istediğiniz tarzda web sitesinin tasarımını yapabilirim.

Duyarlı Tasarımlar

Yaptığım uygulamaları, web sitelerini farklı cihaz boyutlarına duyarlı şekilde (responsive) olarak tasarlayabilirim. Yani masaüstü, mobil veya tablet gibi farklı boyutlardaki cihazların ekran boyutuna göre uygulamanın tasarımının, boyutunun değişmesini, o ekran boyutuna uygun şekilde dizayn edilmesini sağlayabilirim.

publish

İnternete Yayınlama

Oluşturduğum uygulamaları, web sitelerini internet ortamında yayınlayabilirim. Böylelikle sitenizin bir internet adresi olur. O adrese sahip herkes sitenizi görüntüleyebilir. Ayrıca siteyi internete yayınlarken SSL güvenlik sertifikasını da ekleyerek siteyi yayınlarım, böylece sitenin güvenliği sağlanmış olur.

Web Programlama Uygulaması (Eski Web Sitem)

Bu benim sadece HTML/CSS ile yaptığım ilk web sitemdir. Asıl olarak bu siteyi ben hem kendimi geliştirmek için hem de proje ödevi için yaptım. Bu sitede sadece HTML/CSS bilgimi kullandım, içinde herhangi bir programlama dili kullanmadım ve sql veritabanına da bağlamadım.

Bu HTML/CSS ile yaptığım proje veritabanına bağlanmadığı ve dinamik işlemler yapmadığı için sitenin yükümlülükleri azdır, bundan dolayı internettee kolaylıkla yayınlanmıştır , kendim için yaptığım bu ilk siteyi ziyaret edebilir ve hakkımda bilgiler edinebilirsiniz, tabi o bilgiler artık eskidi ve ben artık yeni geliştirdiğim web sitesini kişisel web sitem olarak kullanıyorum. (O kullanacağım yeni sitenin github repository'si: https://github.com/mrtgzn406/My_New_Site )

Sitenin bağlantı adresi: http://muratgozen.cf/
Sitenin github repository'si: https://github.com/mrtgzn406/Web_Programming_Example

İnternet Tabanlı Programlama Uygulaması

Burada ise biz bir internet tabanlı uygulama (web sitesi yaptık) yaptık. Bu sefer Asp.net, C# ve SQL bilgilerimizi harmanlayarak bu projede entegre bir şekilde kullandık, ardından da sitemizi bir hosting servisiyle internete yayınladık.

İsteyen her kullanıcı siteye girip ürün ekleme, silme, güncelleme ve ürünleri listeleme işlemleri yapabilir. Her kullanıcının yaptığı ekleme, silme, güncelleme işlemleri direkt veritabanına kaydedildiği için yapılan değişiklikler, siteyi görüntüleyen diğer kullanıcılara da yansır.

Sitenin bağlantısı: http://reinforcement.somee.com/
Sitenin github repository'si: https://github.com/mrtgzn406/Internet_Based_Programming_Example

Yeni Web Sitem (Şu anda bu sitenin içindesiniz)

Bu ise benim şimdiye kadar edindiğim tüm web tasarım bilgilerimi kullandığım, bütün o bilgileri harmanlayarak meydana getirdiğim bir kişisel web sitesidir. Bu projeyi yaparken HTML, CSS, JavaScript, Sass/Scss, TypeScript, Bootstrap 5 gibi tüm web tasarımıyla âlâkalı bildiğim yazılım bilgilerimi kullandım.

Yakında bu site taslağında bazı hataları gidermeyi ve bu siteyi veritabanı olan bir site haline getirmeyi düşünüyorum. Nihai olarak ise yaptığım bu taslak siteyi kendime göre özelleştirip kendi kişisel web sitem haline getirmeyi düşünüyorum.

Not: Bu site henüz inşa sürecindedir, yapımı devam etmektedir. Yapım tamamlanınca site internette yayınlanacak ve nihai olarak benim kişisel web sitem haline gelecektir.

Sitenin github repository'si: https://github.com/mrtgzn406/My_New_Site

Access Veritabanında Personel Maaşları Takip Programı Uygulaması

Bu projede "Personel Maaşları Takip Programı" yaptık. Projeyi Access veritabanı yönetim sistemini kullanarak yaptık. Projede direkt kullanıma hazırdır, tabloların birbirleriyle bağlantıları, uygulamanın son kullanıcıyıa gösterilecek arayüzüi, uygulamanın nasıl kullanılacağı ile ilgili .docx formatında rapor ve diğer gerekli tüm fonksiyonel ayarlamalar da yapılmıştır. Bu proje de üniversitedeki proje ekibi adına baştan sona benim yaptığım bir projedir.

Not: Bu uygulamayı kullanmak için Microsoft Access yazılımına sahip olmalısınız, ücretli bir yazılımdır.

Uygulamanın github repository'si: https://github.com/mrtgzn406/Access_Database_Example

Sql Veritabanında Personel Maaşları Takip Programı Uygulaması

Bu projede ise "Personel Maaşları Takip Programı"nı MSSQL üzerinden yaptık. Projeyi "Microsoft SQL Server Management Studio" veritabanı yönetim sistemini kullanarak yaptık. Projede tabloların birbirleriyle bağlantıları, veritabanı diyagramı, uygulamanın nasıl kullanılacağı ile ilgili .docx formatında rapor ve diğer gerekli tüm gerekli ayarlamalar da yapılmıştır. Bu proje de üniversitedeki proje ekibi adına baştan sona benim yaptığım bir projedir.

Not: Bu projeyi neredeyse tamamen "Microsoft SQL Server Management Studio" programının tasarım arayüzünden yaptık, daha sonra da "Generate Script" diyerek bu .sql dosyasını elde ettik. Uygulamayı kullanmak için "Microsoft SQL Server Management Studio" programını ve SQL Server'ı kurmanız gereklidir, ikisi de ücretsiz programlardır.

Uygulamanın github repository'si: https://github.com/mrtgzn406/SQL_Database_Example

Mobil Programlama Uygulaması

Bu projede Java dilini kullanarak bir "Gmail Intent" uygulaması yaptım. Bu uygulama bir arayüz uygulaması , yani uygulamada kullanıcının doldurması gereken alıcı alanı, mesaj başlığı alanı ve mesaj alanı gibi temel e-posta alanları var. Kullanıcı bunları doldurup , "Send" butonuna basar, çıkan seçim ekranında "Gmail" uygulamasını seçer ve böylelikle kullanıcının arayüzdeki alanlara yazdığı bütün veriler "Gmail" uygulamasındaki alıcı alanı, mesaj başlığı alanı ve mesaj alanı'na aktarılır.

Uygulamanın github repository'si: https://github.com/mrtgzn406/Mobile_Programming_Example
